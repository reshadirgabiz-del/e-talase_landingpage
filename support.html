2. Mendapatkan Akses Mobile

Aplikasi mobile e-talase tersedia untuk pengguna e-talase yang sudah memiliki akses resmi ke toko.

  1. Buka dashboard web e-talase.
  2. Masuk ke pengaturan akun atau pengaturan aplikasi mobile.
  3. Pilih opsi login mobile.
  4. Scan QR code dari aplikasi mobile atau masukkan kode login.
  5. Pilih toko yang ingin dikelola.
Gunakan QR atau kode login hanya pada perangkat yang Anda percaya. Jika perangkat hilang atau anggota tim tidak lagi berwenang, cabut aksesnya dari dashboard web.

3. Troubleshooting

QR login tidak bekerja

  • Pastikan izin kamera aktif.
  • Pastikan QR code dibuat dari dashboard web produksi.
  • Refresh QR code lalu coba lagi.

Kode login tidak bekerja

  • Pastikan kode belum kedaluwarsa.
  • Masukkan seluruh enam karakter.
  • Buat kode baru dari dashboard web.

Toko tidak muncul

  • Pastikan akun Anda memiliki akses ke toko tersebut.
  • Pastikan toko memiliki paket aktif bila paket aktif diwajibkan.
  • Minta pemilik toko atau administrator memverifikasi peran Anda.

Notifikasi tidak masuk

  • Aktifkan izin notifikasi di pengaturan Android atau iOS.
  • Buka aplikasi lalu simpan preferensi notifikasi dari halaman profil.
  • Pastikan sistem notifikasi backend sudah dikonfigurasi untuk toko.

4. Permintaan Data atau Akun

Untuk meminta akses, koreksi, penghapusan, atau ekspor data pribadi, kirim email ke info@mail.e-talase.com.

Sertakan informasi berikut:

  • Email akun.
  • Nama toko.
  • Jenis permintaan: akses, koreksi, penghapusan, atau ekspor.
  • Konteks data order atau pelanggan yang diperlukan untuk mengidentifikasi permintaan.

Jangan mengirim password, kode login, detail kartu, atau informasi pembayaran sensitif melalui email. Ketentuan pemrosesan data dijelaskan dalam Kebijakan Privasi.
